# Изучение языка программирования Golang

Основы прикладного программирования на языке Go.
- Знакомство с основными функциями языка, написание небольших программ на Go для создания консольных утилит:
  - синтаксис
  - типы данных
  - ввод/вывод данных

- Знакомство с механизмами конкурентности
  - каналы
  - горутины
  - пакет sync
  - инструменты синхронизации (WaitGroup, Mutex и RWMutex)
